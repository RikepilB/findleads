<objective>
Build `lib/places/mapPlaceToLead.ts` — the pure function that maps a validated `RawPlace` to this
app's lead shape, excluding closed businesses (SCRAPE-04) and classifying tier-1 leads whose
`websiteUri` is absent (SCRAPE-05).

Purpose: this is where SCRAPE-04's exclusion-set-not-allowlist design and SCRAPE-05's
signal-not-fact wording both get encoded directly in the data this phase produces, rather than left
for a later UI layer to remember correctly. Depends only on Plan 02-01's `schema.ts` for the
`RawPlace` type — no dependency on `client.ts` or `paginate.ts`, so this plan runs in parallel with
Plan 02-03 in Wave 2.
Output: `lib/places/mapPlaceToLead.ts` and its unit tests.
</objective>
